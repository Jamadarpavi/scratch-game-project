# scratch-game-project
An interactive Scratch project created using Scratch with animations and basic programming logic.  
Scratch Game Project

# Project Description
This is a simple game created using Scratch. The project demonstrates basic coding concepts such as movement, events, and loops.

# Features
- Character movement
- Simple animations
- Score system

# Tools Used
- Scratch

# How to Run
1. Download the .sb3 file.
2. Open it in Scratch.
3. Click the green flag to start the project.
